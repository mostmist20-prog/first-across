How intents work
User-Centric
Crosschain intents are a crosschain limit order plus an action to execute. Intents replace explicit execution steps with user outcomes, allowing relayers to compete to provide the optimal execution path for the user.
Elegant Abstraction
Unlike complex trusted message passing solutions Across connects users to dapps via intents, not just blockchains to blockchains. Developers only need to embed a standard order in protocol actions to create seamless crosschain experiences.
Modular Interoperability
Crosschain intents are powered by a modular system of 3 layers:
  1. Request for quote mechanism
  2. Network of competitive relayers
  3. Settlement layer to escrow user input funds, verify, and repay relayers
